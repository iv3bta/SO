# Operating Systems

A comprehensive collection of Operating Systems projects and implementations exploring fundamental OS concepts through practical bash scripting exercises.

## 📋 Overview

This folder contains laboratory assignments focused on:

- **Bash Scripting Fundamentals** - Shell script development and automation
- **Number Base Conversion** - Converting between decimal and hexadecimal number systems
- **Arithmetic Operations** - Basic calculator functionality with multiple operations
- **Modular Programming** - Code organization through function libraries and script sourcing
- **User Interaction** - Menu-driven programs with interactive input handling
- **Process Management** - Script execution and argument passing between programs

## 🛠️ Technologies

- **Language:** Bash Shell Script
- **Platform:** Linux
- **Concepts:** Functions, Conditionals, Case statements, User Input, Number Systems

- **laboratory-3/** - Calculator and number conversion system with menu-based operations
